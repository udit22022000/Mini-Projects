CODE EXPLANATION
              CODE :

              var greeting = "Hello"
              var name = "Titans"
  
              function sayHello(){
              console.log(greeting + " " + name)
              }
  
              sayHello();

STEP 1: First of all a global execution context is created. Thereafter starts the most initial phase called creation phase, in this phase all the variables and functions are declared in the Global Execution context.


STEP 2: Now the Execution phase starts, where all variables are assigned values and lines are being executed one by one as encountered.


STEP 3: As we can see that, in the last line of the code, a function sayHello is being called. So now a brand new functional execution context will be created, then there will be 2 phases of execution for that partcular function. The first phase will be creation phase, where all the variables inside it will be declared.


STEP 4: Now the execution phase starts inside the functional execution context of sayHello function, where all the lines inside the function is executed one by one.
