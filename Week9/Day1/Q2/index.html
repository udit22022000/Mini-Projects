As we know that arrays are special types of object which stores values in numerically indexed fashion. Array Destructuring comes into picture when we need to define multiple variables at one go. We can define as many variables we want in one line using array destructuring.
Process goes like this, on the Left hand side write all the variable names that we want, within an array and on the right hand side, put all the values that we want to get assigned to those variables inside an array. Values will get allocated to the variables with respect to their index i.e variable at index x on L.H.S will be assigned value at index x on R.H.S.
We can define bunch of variables at one go using object destructuring.
Process goes like this, on the left hand side write all the variables names that we want, inside an object and on the right hand side define an object having key names same as variable names and assign values to the keys accordingly. Like this all the variables will get defined with corresponding values associated with the keys.
Spread operator is mostly used in places where more than 1 values are expected. Suppose, if we want to append all the elements of a particular array to another array, we can use spread operator to append all at once,spread opeartor just spreads out all the values inside it.
It is basically the collection of all the remaining elements in the form of an array.
Rest Operator is mostly used in function parameters, where we don't know how many arguments will be passed by the user, in those cases all the values that are passed as arguments, are stored in an array.
Default values are generally assigned to the varaibles, in case value is not being assigned to that variable, at that time default value assigned to that variable. It is generally used while variables are assigned values through array or object destructing.
