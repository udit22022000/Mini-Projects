ARROW FUNCTIONS
[NOTE: Click upon the below mentioned topic names to see the explanation]
Arrow Functions
For Each
Map
Reduce
Filter
Uses :
1. They have a shorter & handy syntax.
2. They have an option to return the statements implicitly.
3. If we use an arrow functions inside another function, the arrow functions retains the scope of the method in which it was declared.

Limitations :
1. They are not suitable for methods inside an object.
2. They can not be used as constructor.
3. They doesn't have any name, they are generally called anonymous functions.
For each is a higher order function in javascript. Generally it is used as an alternative in place of loops. It takes an array and a function as input. After that, for each element of the array, forEach executes that particular function.
Map is a higher order function in javascript.It is almost similar to forEach. It takes an array and a function as input. After that, for each element of the array, map executes that function and stores the value generated each time in an array and at last returns the whole array.
Reduce is a higher order function in javascript. It takes an array, function and a comparator as input. It traverses through each element of the array and uses the output generated from the previous one as a input for the next element for each and every element of the array. At last it returns the output generated from the final element.
Filter is a higher order function in javascript. It takes an array and a function as input. After that, filter takes all the elements one by one and executes that particular function. Filter creates a new array and stores all those elements for which a function returns true and at last filter returns that array as output.
